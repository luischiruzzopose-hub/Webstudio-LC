# WebstudioLC

Página web profesional para LC Web Studio.

## Archivos incluidos

- `index.html`: estructura principal de la web.
- `styles.css`: diseño visual, colores, responsive y animaciones.
- `logo.png`: logo principal integrado en la página.
- `logo.svg`: versión vectorial simple del logo.

## Cómo ver la página en tu computadora

1. Abrí la carpeta `WebstudioLC_completo`.
2. Hacé doble clic en `index.html`.
3. Se abrirá en tu navegador.

## Cómo subirla a GitHub

Abrí la terminal dentro de esta carpeta y pegá:

```bash
git init
git add .
git commit -m "Primera version de la web"
git branch -M main
git remote add origin https://github.com/luischiruzzopose-hub/WebstudioLC.git
git push -u origin main
```

## Cómo publicarla en Vercel

1. Entrá a https://vercel.com
2. Iniciá sesión.
3. Clic en `Add New` > `Project`.
4. Elegí el repositorio `WebstudioLC`.
5. Clic en `Deploy`.

Vercel te va a dar un enlace público para compartir con clientes.
